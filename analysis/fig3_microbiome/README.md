# Figure 3 — Microbiota composition is unaffected by ST1.75 colonization

## Overview
16S sequencing analysis comparing ST1.75, ST1.12, and uninfected mice
across Days 0-7. Shows protection is independent of microbiota composition.

## Script
`microbiome_analysis.m` — stacked bars, alpha diversity, PCoA,
Clostridioides abundance

## Data
`data/raw/16s_sequencing/tblAbund.xls` — genus-level relative abundance
(119 samples, Days 0-7)

## How to run
Set MATLAB working directory to `data/raw/16s_sequencing/` then run script

## Dependencies
MATLAB R2025b, Statistics and Machine Learning Toolbox
